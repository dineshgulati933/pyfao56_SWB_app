Welcome to the FAO-56 Soil Water Balance App

This application allows you to simulate soil water balance based on FAO-56 guidelines. Follow these steps:

  1. Provide plant parameters such as crop coefficients and growth stages.
  2. Enter weather data, either by uploading a file or fetching gridMET data.
  3. Define soil characteristics, including field capacity and wilting point.
  4. Input irrigation details, manually, through file upload or set auto irrigation.
  5. Run the simulation to generate results with graphical and tabular outputs.
